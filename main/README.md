# ims

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### 项目三方库
- element-ui ui组件
- lodash js工具库
- axios http请求
- iconfont 图标库
